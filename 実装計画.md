# 相性占い分析アプリ 実装計画

---

## 全体方針

- StitchのUIデザイン（4画面）をベースにFlaskアプリとして実装
- 占い・分析手法一覧の39手法のうち、**生年月日から自動計算可能なものを優先実装**
- 個人用ローカルアプリ（localhost実行）

---

## 技術スタック

| レイヤー | 技術 |
|---|---|
| バックエンド | Python 3 + Flask |
| テンプレート | Jinja2（Flask標準） |
| フロントエンド | Stitch生成HTML + Tailwind CSS (CDN) + Chart.js (CDN) |
| 占星術計算 | kerykeion（Swiss Ephemeris） |
| 旧暦変換 | cnlunar |
| データ保存 | JSON（個人用） |

## 各計算エンジンの入出力仕様

### 共通インターフェース

全ての calculator は以下の共通インターフェースに従う：

```python
class BaseCalculator:
    def calculate(self, person_a: dict, person_b: dict) -> dict:
        """
        Args:
            person_a: {name, birthday, birth_time, birthplace, mbti, blood_type, enneagram, wing}
            person_b: {name, birthday, birth_time, birthplace, mbti, blood_type, enneagram, wing}
        
        Returns: {
            "name": "数秘術",           # 手法名
            "category": "numerology",   # カテゴリID
            "score": 5,                 # 1-5のスコア（★の数）
            "score_100": 95,            # 100点満点スコア
            "summary": "...",           # 一行サマリ
            "details": {                # 詳細データ（手法ごとに異なる）
                "person_a": {...},
                "person_b": {...},
                "compatibility": {...}
            },
            "highlights": ["..."],      # ハイライト（箇条書き）
            "advice": "...",            # アドバイス
        }
        """
```
